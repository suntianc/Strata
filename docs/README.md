## 🎯 快速查找

### 我是新用户,如何开始?

1. 阅读 [快速启动指南](../QUICKSTART.md) (5 分钟)
2. 配置 AI 服务: [LLM 配置指南](../LLM_CONFIG_GUIDE.md)
3. 查看 [用户手册](./用户手册.md) 了解详细功能

### AI 对话不工作怎么办?

1. 查看 [故障排查指南](../TROUBLESHOOTING.md)
2. 检查 [LLM 配置指南](../LLM_CONFIG_GUIDE.md)
3. 尝试使用 Ollama (无 CORS 问题)

### 我想了解技术实现细节

1. 阅读 [架构概览](../ARCHITECTURE_SUMMARY.md)
2. 查看 [实施指南](../IMPLEMENTATION_GUIDE.md)
3. 参考 [详细设计文档](./详细设计文档.md)

### 我想贡献代码

1. 阅读 [实施指南](../IMPLEMENTATION_GUIDE.md)
2. 查看 [架构概览](../ARCHITECTURE_SUMMARY.md)
3. 提交 Pull Request
